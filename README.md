# 工厂车间管理系统 java336

本项目是基于 Spring Boot 和 Vue 技术栈开发的工厂车间管理系统，旨在提高工厂生产管理的效率与信息化水平。

## 技术栈

- 后端：Spring Boot
- 前端：Vue + ElementUI
- 其他：Node.js

## 系统角色

- 管理员：拥有个人中心、人员管理、看板信息管理、设备信息管理、生产开立管理、生产工序管理、生产流程管理等权限。
- 人员：具备登录、查看管理员发布信息的能力。

## 运行环境

- JDK 1.8
- IntelliJ IDEA
- MySQL
- Visual Studio Code

## 功能模块

### 管理员

- 个人中心
- 人员管理
- 看板信息管理
- 设备信息管理
- 生产开立管理
- 生产工序管理
- 生产流程管理

### 人员

- 登录
- 查看发布信息

## 核心亮点

- 前后端不分离，易于维护与部署。
- 使用 Spring Boot 和 Vue，保证了系统的高效性和响应速度。
- 基于角色权限管理，确保了系统安全性和数据保密性。
